HTML 문법

앞으로 여기에 적용할 문법도 함께 정리합니다

[자주 사용하는 태그]
- strong : 강조
- u : 밑줄
- br : 줄바꿈

- p : 단락표시 -> 정보로서 좀 더 가치 있는 HTML이 됩니다. 정해진 여백만큼만 띄어짐, 시각적으로 "br"보다 떨어짐

"br"보다 자유도가 떨어지는 부분을 css로 보완함. style라는 HTML이라는 속성을 사용하고 그 속성값에 css라는 HTML과는 완전히 다른 컴퓨터 언어의 문법이 들어오게 약속이 되어 있습니다. "px" 값에 따라 위쪽 넓이가 달라집니다.

- img : 이미지 표현
- Attribute : 태그가 태그의 이름만으로는 정보가 부족할 때 이런 속성을 통해서 더 많은 의미를 부가할 수 잇게 됩니다.
ex) img 태그 다음에 사용되는 Attribute "src = 표현할 이미지, width = 이미지 사이즈"

- li : list의 약자
- ul : Unordered List의 약자, li 태그를 그룹화 시키는 부모태그
- ol : Ordered List의 약자, li 태그를 그룹화 시키면서 자동넘버링 됨
- a : anchor의 약자(정보의 바다에 닻을 내린다는 시적인 표현) HyperText를 의미하고 link를 걸대 사용
